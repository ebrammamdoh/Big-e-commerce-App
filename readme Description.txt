this e-commerce app is build in Angular 5 Tempelate-forms technology as a front-end.
with asp.net API as a back-end and SQL server database.

it manages the Admin and the ordinary User with login and logout based on Json Web Token the server created.

the Admin cabable of reviewing products, add product, edit product, delete product,
and the User who can reviews prducts and search by name or by category 
and also buy specific product to be added to his shopping cart.
