Welcome
this e-commerce app is built in Angular 5 Template-forms technology as a front-end.
with asp.net API as a back-end and SQL server database.

it manages the Admin and the ordinary User with login and logout based on JSON Web Token the server created.
the Admin capable of reviewing products, add products, edit products, delete products,
and the User who can reviews products and search by name or by category 
and also buy a specific product to be added to his shopping cart.
